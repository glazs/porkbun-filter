Porkbun Search Price Filter
===========================

This UNOFFICIAL extension filters out domains at porkbun.com search results that are too expensive to purchase or renew. 

Installation from marketplaces
------------------------------

### Chrome Web Store

<https://chromewebstore.google.com/detail/porkbun-search-price-filt/lfimgflmgnoijahleiiebdjgnmjmbljc>

### Firefox Add-ons

<https://addons.mozilla.org/en-US/firefox/addon/porkbun-search-price-filter/>

Manual installation from source
-------------------------------

0. Download the extension source code.

### Chrome / Chromium

1. Visit **chrome://extensions** in your browser 
   (or open the Chrome menu by clicking
   the icon to the far right of the address bar:
   the icon is three vertical dots.
   Then select **Extensions** under the **More Tools** menu
   to get to the same place).
2. Ensure that the **Developer mode** toggle
   in the top right-hand corner is turned on.
3. Click **Load unpacked**
   to open a file-selection dialog.
4. Navigate to the directory where your
   extension files are located and select the **src** directory.

### Firefox

1. Visit **about:debugging** in your browser.
2. Click **This Firefox** in the sidebar.
3. Click **Load Temporary Add-on...**.
4. Navigate to the directory where your extension files
   are located and select the **manifest.json** file
   inside the **src** directory.
